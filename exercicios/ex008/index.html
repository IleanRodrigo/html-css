Principais Formatações

Negrito / Destaque

A forma antiga de criar um termo em destaque é usando a tag <b>.

A nova forma de criar um termo em destaque é usando a tag <strong>.

Itálico / Ênfase

A forma antiga de criar um termo em ênfase é usando a tag <i>.

A nova forma de criar um termo em ênfase é usando a tag <em>.

#Dica de Mestre: Você também pode usar um atalho para "envelopar" uma frase dentro de uma tag, muito útil quando você quer colocar uma tag numa frase já feita. Basta selecionar a frase em si e apertar Ctrl+Shift+P que aparecerá uma janela onde você deverá buscar por >abb, daí é só clicar na opção "Emmet: Envelope com a Abreviatura". Ele te dará uma caixa de texto onde você poderá digitar a tag que desejar (digite a tag sem os colchetes). Ao pressionar Enter imediatamente a frase selecionada ficará dentro da tag escolhida.

Texto Marcado

Podemos criar também um texto marcado usando a tag <mark>.

Texto grande e pequeno

Podemos deixar um texto um pouco maior usando a tag <big>. Porém essa tag está depreciada pela HTML5 e daqui à alguns anos pode ser totalmente excluída.

Já se queremos deixar um texto um pouco menor, podemos usar a tag <small>. Diferente da tag <big>, essa tag continua ativa na HTML5. Muito provável que isso se deva ao fato dela ser usada na escrita das "letras pequenas" de um documento.

Texto deletado e Texto inserido

Para marcar um texto como excluído e indicar que ele pode ser lido, porém desconsiderado, usamos a tag <del>.

Para marcar um texto como inserido e indicar que ele deve ser lido e levado em consideração, usamos a tag <ins>.

Antes, usávamos as tags <strike> para Deletado e <u> para Inserido.

Texto Sobrescrito e Texto Subscrito

Caso queirámos escrever algo como x2 colocando o 2 ali em cima, utilizamos a tag <sup>.

E se quisermos escrever algo como H2O colocando o 2 ali embaixo, ultilizamos a tag <sub>.
