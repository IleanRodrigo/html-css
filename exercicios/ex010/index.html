Usando links externos

Link Externo
É um link que aponta pra outro site que não é nosso.
Tag <a>
É a tag anchor ou âncora, capaz de criar um hyperlink.
Parâmetro href
O parâmetro de referência a hypertexto (hypertext reference) é praticamente obrigatório e é nele que colocamos a URL do destino que desejamos.
Parâmetro target
O parâmetro target (alvo) é capaz de especificar de que forma será aberto o hyperlink em questão. Por padrão, o valor presente no parâmetro é _self (mesmo ou próprio) que abre o link na mesma aba. Para que a abertura seja feita em uma outra aba, usamos o valor _blank (em branco. No caso, janela em branco).
Parâmetro rel
Esse parâmetro indicar qual a "natureza" do destino, e possui valores como:
  • Next (próxima parte do documento atual)
  • Prev (parte anterior do documento atual)
  • Author (site do author do artigo atual)
  • External (para outro site que não faz parte do site atual)
  • Nofollow (para um site não endossado. Quando você faz um link pra matéria de um site que você não sabe se é o oficial. Ou então sempre que você fizer um link patrocinado de uma empresa que pagou pra estar ali. Basicamente você que não dá aval para o conteúdo e diz para o Google "não seguir" Nofollow)
A importância desse parâmetro está em facilitar o mecânismo de busca a encontrar o sistema de navegação que você tá criando pro seu site. Você diz para o Google quais são as próximas páginas e as anteriores.

Exemplo:

Você pode acessar o meu repositório público no GitHub

Acesse também o meu canal gratuito no YouTube

Usando links internos

Link Interno
É um link que faz uma relação com outra página que está dentro do nosso servidor e diz respeito a nossa própria página.

Exemplo:

Muito importante! Veja esta mensagem.

Acesse também a nossa página de notícias
