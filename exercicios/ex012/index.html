Este vídeo está hospedado no meu próprio servidor.

Quanto mais sources você colocar, maior será a compatibilidade para vários navegadores, e portanto, para vários usuários.

Inserindo vídeos do YouTube

Diferente do vídeo hospedado localmente que consome dados do servidor ao ser reproduzido, os vídeos hospedados externamente consomem dados dos servidores externo, ou seja, vídeos hospedados no YouTube quando reproduzidos no seu site consomem dados dos servidores do YouTube.
